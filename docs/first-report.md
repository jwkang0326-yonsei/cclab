**작성일:** 2026.01.18 (Updated: 2026.01.19)
**작성자:** 기획팀
**프로젝트명:** WithBible (가칭)

## 1. 프로젝트 개요 (Overview)

### 1.1. 배경 및 목적

- **배경:** 청년(20대)부터 장년(40대)까지 이어지는 '청장년부'는 직장과 육아 등으로 가장 바쁜 세대임. 1년 1독의 의지는 있으나, 현실적인 시간 부족과 동기부여 부재로 중도 포기율이 높음.
- **목적:** 개인의 부담을 줄여주는 **'심플한 인증'**과 공동체가 함께 달성하는 **'시각적 성취감'**을 결합하여, 바쁜 일상 속에서도 성경 읽기를 지속하도록 도움.

### 1.2. 핵심 가치 (Core Value)

- **Habit:** 매일 짧은 시간 투자로 거룩한 습관 형성.
- **Connection:** 각자 흩어져 살지만 말씀 안에서 연결됨을 확인.
- **Visual:** 읽은 결과가 눈에 보이는 확실한 성취감 제공.

### 1.3. 타겟 및 플랫폼

- **타겟:** 스마트폰 활용에 능숙하며, **효율적 시간 관리**와 **세련된 감각**을 선호하는 2040 청장년부.
- **플랫폼:** 하이브리드 모바일 앱 (Android / iOS) - 접근성 및 리마인더 기능 강화.

### 1.4. 운영 및 관리 정책 (Governance) **[NEW]**

- **폐쇄형 커뮤니티:** 신뢰할 수 있는 데이터 관리를 위해 교회 등록은 '승인제'로 운영.
- **역할 분리:** 전체 시스템 관리, 개별 교회 운영, 소그룹 관리, 일반 활동의 권한을 명확히 분리.
- **데이터 주권:** 각 교회의 성도 데이터 및 통계는 해당 교회 관리자만 열람 가능하도록 보안 설계.

## 2. 사용자 페르소나 (Target Persona)

이 그룹은 **"시간 효율성"**과 **"확실한 보상(성취감)"**이 가장 중요합니다.

| 구분 | 내용 |
| --- | --- |
| **이름/나이** | **이믿음 (38세, 워킹맘/직장인)** |
| **상황** | 퇴근 후 육아와 집안일을 마치면 밤 10시. 녹초가 되어 성경 책을 펴기가 무거움. |
| **Pain Point** | "마음은 굴뚝같은데 짬이 안 나요. 며칠 밀리면 죄책감 때문에 아예 놓게 되더라고요." |
| **Needs** | "출퇴근길이나 자기 전 **5분 안에 끝낼 수 있는 심플한 방식**이면 좋겠어요. 그리고 우리 구역(셀) 식구들이랑 서로 격려하는 기능이 있으면 억지로라도 할 것 같아요." |

## 3. 서비스 프로세스 (User Flow)

1. **진입 (Onboarding & Authorization)**
    - 로그인 (Social Login)
    - **교회 연결:** 교회에서 발급받은 **'초대 코드'** 입력 → 교회 자동 매칭 **[변경]**
    - 소속 그룹(구역/셀) 선택 및 개인 목표 설정
2. **활동 (Daily Action) - "Simple & Fast"**
    - [알림] 출근/점심/취침 시간 등 맞춤형 알림 수신
    - [인증] 범위 체크 → **핵심 요약 퀴즈(부담 없는 난이도)** → 한 줄 묵상
    - [성장] 나의 기여로 팀의 나무가 자라거나 건물이 지어짐
3. **소통 (Social Interaction) - "Polite Nudge"**
    - 그룹 달성률 그래프 확인 (선의의 경쟁)
    - **'함께해요(응원)'** 보내기 (너무 가벼운 '찌르기' 대신 정중한 격려 알림)
    - 동료의 묵상에 '아멘', '기도해요' 이모지 반응

## 4. 주요 기능 명세 (Functional Requirements)

*40대 사용자까지 고려하여 UI/UX는 최대한 직관적이고 글씨는 가독성 있게 설계*

| 분류 | 기능명 | 상세 설명 | 사용자 |
| --- | --- | --- | --- |
| **인증** | **원터치 기록** | 복잡한 입력 없이 터치 한 번으로 읽음 처리 (진도율 자동 계산) | 성도 |
|  | **핵심 퀴즈** | 본문 이해를 돕는 간단한 퀴즈 (틀려도 패널티 없음, 재도전 가능) | 성도 |
|  | **묵상 나눔** | 긴 글쓰기가 부담스러운 이들을 위한 '키워드 선택' or '짧은 한 줄' | 성도 |
| **시각화** | **공동체 비전** | 개인의 읽기가 모여 교회의 비전(성전 건축, 선교 지도 등)이 완성됨 | 전체 |
|  | **직관적 통계** | "올해 성경의 15%를 읽으셨어요!" 등 수치와 그래프로 명확히 표시 | 성도 |
| **소셜** | **그룹 랭킹** | 구역/셀 대항전 (개인 성적은 비공개, 그룹 평균만 공개) | 전체 |
|  | **스마트 알림** | 미참여 기간이 길어질 때 격려 문구 발송 ("집사님, 잠시 쉬어가도 괜찮아요!") | 시스템 |
| **관리** | **그룹 관리** | 셀/구역 편성, 리더 임명, 그룹원 이동 | **교회 관리자** |
|  | **통계 대시보드** | 교회 전체 통독 현황, 참여율 저조 그룹 파악 | **교회 관리자** |

## 5. 디자인 컨셉 무드보드 (Design Concept)

*2040 세대를 아우르기 위해 유치하지 않고 '세련된' 톤앤매너 유지*

### **Concept A. 모던 가드닝 (Modern Gardening) [추천]**

- **컨셉:** "우리의 말씀 생활이 모여 숲을 이룹니다"
- **스타일:** 킨포크(Kinfolk) 스타일의 감성적인 식물 일러스트, 편안한 그린 & 우드 톤.
- **장점:** 20대부터 40대까지 가장 호불호가 적고, 바쁜 일상 속 '힐링'의 느낌을 줌.

## 6. 향후 일정 및 로드맵

### Phase 1: MVP (Mobile App Focus)
*   **목표:** 핵심 기능(읽기, 인증, 랭킹) 및 앱 안정성 확보.
*   **관리자:** 앱 내 '관리자 모드' 미지원. 초기 교회 데이터는 개발팀(Super Admin)이 DB 직접 주입.
*   **플랫폼:** Android / iOS

### Phase 2: Admin Expansion (Web)
*   **목표:** 교역자/간사님이 직접 운영 가능한 환경 구축.
*   **기능:** PC 웹 관리자 페이지 (통계 다운로드, 조직편성, 주보 텍스트 생성).
*   **확장:** 교회별 커스텀 공지사항 등.
